# babylonjs-vrm-loader

Refer to https://github.com/virtual-cast/babylon-vrm-loader

## Quick Start

```sh
pnpm add @babylonjs/core @babylonjs/loaders babylonjs-vrm-loader
```

```ts
import 'babylonjs-vrm-loader';
```

## Features

- [x] Supports Babylon.js 6.x
- [ ] Supports VRM 1.0
